🪙 What are FI-Tokens?

FI-Tokens are the fuel for FieldInsight's AI agents. Every AI action — a Voice Agent transcription, a Document Agent invoice parse, a Booking Agent call, a Dispatch Agent assignment — consumes a small number of tokens. Your plan includes a generous monthly allocation, and if you exceed it you can purchase top-ups at AU $0.005 per token.

  • Voice Agent note — ~200 tokens per job note
  • Booking Agent inbound call — ~800 tokens per booking
  • Document Agent invoice parse — ~500 tokens per PDF
  • Dispatch Agent auto-assignment — ~150 tokens per job
  • Inbox Agent email classification + reply draft — ~300 tokens
